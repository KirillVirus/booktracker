Описание программы
Random Quote Generator — это десктопное приложение с графическим интерфейсом для генерации вдохновляющих цитат. Программа позволяет пользователю получать случайные цитаты из предопределённой коллекции, сохранять историю просмотренных цитат, фильтровать их по автору и теме, а также пополнять коллекцию собственными цитатами.

Что решает программа:

Помогает быстро получить порцию вдохновения или мотивации

Сохраняет историю, чтобы не забыть понравившиеся мысли

Позволяет систематизировать цитаты по авторам и темам

Даёт возможность создать собственную коллекцию цитат

Основные возможности:

🎲 Генерация случайных цитат

📜 История просмотров с временными метками

🔍 Фильтрация истории по автору и теме

➕ Добавление новых цитат в коллекцию

💾 Автоматическое сохранение данных в JSON

Требования для запуска
Что нужно установить на компьютер:

Python 3.6 или выше — скачать с python.org

Библиотеки (все входят в стандартную поставку Python, дополнительная установка не требуется):

tkinter — для графического интерфейса

json — для работы с файлом данных

random — для случайного выбора цитат

datetime — для временных меток

Как запустить
Способ 1: Через командную строку
bash

git clone https://github.com/ivanpetrov/random-quote-generator.git

# Перейти в папку проекта
cd random-quote-generator

# Запустить программу
python main.py
Способ 2: Двойным щелчком
Скачай папку с проектом

Убедись, что файл называется main.py

Дважды кликни по main.py — программа запустится

Как пользоваться
Главное окно программы
После запуска откроется окно с несколькими областями:

Верхняя область — здесь отображается сгенерированная цитата

Кнопка «Сгенерировать цитату» — нажми для получения случайной цитаты

Фильтры — выбери автора или тему, чтобы отфильтровать историю

Список истории — все сгенерированные цитаты с датой и временем

Форма добавления — поля для добавления новой цитаты

Кнопка «Очистить историю» — удаляет всю историю

Пошаговая инструкция
Шаг 1: Сгенерировать цитату
Нажми кнопку «✨ Сгенерировать цитату» — в верхней области появится случайная цитата.

Шаг 2: Посмотреть историю
Сгенерированная цитата автоматически появится в списке истории с указанием времени.

Шаг 3: Отфильтровать историю
Выбери автора из выпадающего списка — останутся только цитаты этого автора

Выбери тему — останутся только цитаты на эту тему

Нажми «Сбросить фильтры» — вернётся полная история

Шаг 4: Добавить свою цитату
Введи текст цитаты

Введи имя автора

Введи тему (например, «мотивация», «жизнь», «успех»)

Нажми «Сохранить цитату»

Шаг 5: Очистить историю
Нажми «🗑️ Очистить историю» и подтверди действие.
